# FinRAG - Financial Retrieval-Augmented Generation

An advanced RAG system built on RAPTOR's hierarchical tree structure, optimized for financial documents.

## Quick Start

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set your OpenAI API key:
```bash
# Windows PowerShell
$env:OPENAI_API_KEY="your-api-key-here"
```

3. Run the example:
```bash
python example.py
```

## Usage

```python
from finrag import FinRAG
from config import FinRAGConfig

# Initialize
config = FinRAGConfig(openai_api_key="your-key")
finrag = FinRAG(config)

# Add documents
text = finrag.load_pdf("report.pdf")
finrag.add_documents([text])

# Query
result = finrag.query("What is the revenue?")
print(result['answer'])
```
